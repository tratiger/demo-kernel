# demo-kernel

## 概要
本プロジェクトは、i686（32bit）アーキテクチャをターゲットとし、現代のLinuxの基本概念を構築するフルスクラッチOS開発です。
Jules環境での自動テスト（CI/CD的な開発）を可能にするため、シリアルポート（UART）によるテキスト入出力を最優先のインフラとして構築します。

## 全体章立て（各章に10〜15のステップが存在）
* **第1章: 最速生存確認とシリアルログ基盤の構築** 
* **第2章: CPU基礎の確立 (GDTとTSSの完全理解)**
* **第3章: 割り込み（IDT）とピュアなハードウェア制御（タイマー・キーボード）**
* **第4章: 物理・仮想メモリ管理 (2段階ページングの固定化)**
* **第5章: カーネルヒープ（動的メモリ割り当て）の解放**
* **第6章: カーネルスレッドとスケジューラ（マルチタスクの心臓部）**
* **第7章: ユーザーモード（Ring 3）への移行とメモリ孤立**
* **第8章: システムコールインフラ（int 0x80）の敷設**
* **第9章: 仮想ファイルシステム（VFS）と初期RAMディスク（initrd）**
* **第10章: ユーザー空間での自作シェルの動作**
